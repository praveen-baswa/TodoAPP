# TodoAPP

This project is using Angular framework v20.3.10

Following functionalities added to this front end project

* Show all items as a list 
* Delete an item from the front end
* Add an item from the front end

<b>NOTE:</b> This project requires the API url. <br>
e.g: https://localhost:7066/api <br>
If the API is running on different port in your system please change at the following location in the front end application.<br>
URL is available in the file 'environment.ts'
